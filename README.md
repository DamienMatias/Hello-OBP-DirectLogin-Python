Python Example for OBP API
==========================

## Setup

```bash
$ pip install -r requirements.txt
```


## Hello OBP

```bash
#compatible with python2 and 3
$ python hello_obp.py
```


## Hello Payments

```bash
$ python2 hello_payments.py
```


## Import Sandbox

Edit the data to import in the script, then run:

```bash
$ python2 import_sandbox.py
```
